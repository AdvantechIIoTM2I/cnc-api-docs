# Availability Analysis APIs
---

## Information
* Availability Analysis APIs compute Machines' availability and use different ways of evaluation to display it. 
* The computed value includes
  * Avaliability
  * Run / Down / Idle / Off time
  * OFF count and  Run+Idle+Down total time
* The following API compute total Availability of one to multiple devices:
  * [MAvail](MAvail.md)
* The following API returns availability trend of one to multiple devices:
  * [SMAvailTrend](SMAvailTrend.md)
* The following API returns daily availability information:
  * [SMAvailDaily](SMAvailDaily.md)
